1. Авторизация / Регистрация
- Регистрация нового пользователя (обязательные поля, валидация email)
- Вход с валидными 
- Невалидными данными
- Восстановление пароля
- Log out

2. Каталог товаров
- Отображение списка товаров (изображение, цена, название, рейтинг)
- Сортировка (по цене, популярности, новизне)
- Фильтрация (по категориям, брендам, цене, размеру и т.д.)
- Карточка товара (фото, название, описание, количество, кнопка «В корзину»)

3. Поиск
- Поиск по существующему / несуществующему слову
- Поиск с опечатками (если есть подсказки)
- Пустой поиск
- Очистка поиска

4. Корзина
- Добавление товара (с выбором количества, размера, цвета)
- Изменение количества в корзине
- Удаление товара из корзины
- Пересчёт итоговой суммы
- Переход к оформлению заказа

5. Оформление заказа (Checkout)
- Заполнение данных доставки (обязательные поля, маски ввода)
- Выбор способа доставки и оплаты
- Подтверждение заказа
- Проверка email-уведомления (если есть)
- Ограничения: пустая корзина, незаполненные поля
